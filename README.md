## Tiralabra

## TiranReitit

TiranReitit on lyhyimmän/nopeimman reitinhakusovellus. Käyttäjä voi syöttää valitsemansa aloituspisteen sekä määränpään ja sovellus tulostaa käyttäjälle lyhyimmän/nopeimman reitin.
